# 🏨 Hospitality Data Analysis Project

📖 Description

This project focuses on analyzing hospitality industry data to derive meaningful insights on revenue, bookings, occupancy rate, and customer behavior. The analysis is performed using SQL, Excel, Power BI, and Tableau to support data-driven decision-making.

## 🎯 Key Features

* Data cleaning and preprocessing using Excel
* SQL-based KPI calculations and analysis
* Interactive dashboards using Power BI and Tableau
* Business insights on revenue, occupancy, and booking trends


## 🛠️ Tech Stack

* SQL
* Excel
* Power BI
* Tableau

## 📂 Project Files

* `Hospitality_Sql.sql` – SQL queries for KPI calculations
* `Hospitality Dashboard.xlsx` – Data cleaning and preprocessing
* `hospitality analysis pb.pbix` – Power BI dashboard
* `Hospitality_tableau.twbx` – Tableau dashboard

## 📊 Key KPIs Analyzed

* Total Revenue
* Total Bookings
* Occupancy Rate (%)
* Total No Shows
* Total Checkouts
* Booking Status Distribution
* 
## 🚀 How to Use

1. Open SQL file and run queries in MySQL / any SQL tool
2. Use Excel file for raw data understanding
3. Open Power BI file (.pbix) to view dashboard
4. Open Tableau file (.twbx) for visualization

## 📈 Insights

* Revenue trends help identify peak business periods
* Occupancy rate shows hotel performance efficiency
* No-show analysis helps improve booking strategies
* Booking status distribution gives customer behavior insight
